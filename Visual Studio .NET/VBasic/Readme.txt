TeeChart Pro AX 
===============

Upgrading projects to Visual Studio .NET

The projects in this folder have been upgraded
from Visual Studio 6 VB projects. Automatically
generated upgrade comments have been left in 
the code. All projects upgraded without issue
except for the Multiple Axes project that
required a tee file copy of the Chart to be 
made before upgrade, the Chart was then 
replaced by an empty Chart and after import
the tee file was re-imported.
